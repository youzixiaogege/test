## 项目名称 Shooping

# 编码测试与打包发布项目：

1. 编码测试：
   yarn add run dev
2. 打包发布:
   yarn add run build
   yarn add install -g serve
   serve dist

## 项目结构：

APi: 与后台交互模块文件夹
assets： 通过资源文件夹 比如 img 文件，css 样式
components： 组件
filters: 自定义过滤器模块文件夹
mock： 模拟数据接口文件
views： 路由组件文件夹
router： 路由文件夹
store： vuex 相关的模块文件夹
App.vue: 应用组件
main.js ： 入口 js
